# Frost

Open Source Alternative to Vercel, Netlify, Railway, Render and Neon.

## Overview

- **Projects** - Group related services with shared environment variables
- **Services** - Deploy from git repos or Docker images
- **Domains** - Attach custom domains with automatic SSL

## Getting Started

1. [Install Frost](/docs/installation) on your server
2. Create a project
3. Add a service
4. Deploy
